# 🤖 AI Resume Assistant

An AI-powered Resume Assistant built using Python and Streamlit that helps users analyze, improve, and optimize resumes for better ATS (Applicant Tracking System) performance. The application uses Generative AI to provide smart suggestions, resume insights, and professional feedback.

---

## 🚀 Features

- 📄 Upload Resume in PDF Format
- 🤖 AI-Based Resume Analysis
- 📊 ATS Score & Resume Evaluation
- 💡 Skill Improvement Suggestions
- 🧠 Generative AI Powered Assistance
- 🔍 Resume Content Understanding
- ⚡ Interactive and User-Friendly UI

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- Mistral AI
- ChromaDB
- PyPDFLoader
- Vector Embeddings
